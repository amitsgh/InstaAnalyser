## Quick start
<br>

Clone this repo

```
git clone https://github.com/amitsgh/InstaAnalyser.git
cd InstaAnalyser-main
```

#### Install Libraries

```
pip install -r requirements.txt
```

### Start Streamlit

```
python main.py
```

- Open Local URL: `http://localhost:8501` or Network URL: `http://192.168.0.102:8501` in browser and use
